# サンプルCSV

`/dashboard/import-export` で使うサンプルCSVです。

- `customers.csv`
- `staff.csv`
- `services.csv`
- `reservations.csv`
- `peakmanager-customers-sample.csv`

予約CSVは、同名のスタッフ・メニュー・ブースが先に存在する前提です。まず `staff.csv` と `services.csv` を取り込んでから試すと確認しやすくなります。
PeakManager 顧客明細CSVは `peakmanager-customers-sample.csv` で表示イメージを先に確認できます。
