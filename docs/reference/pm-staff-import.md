## 設計メモ（ユーザー指示）
- スタッフ詳細（名前クリックで開く編集画面）の項目は **PM 錦糸町の詳細画面** を参照して作る。
  - 項目: ID / 所属店舗 / 社員番号 / 名前 / 姓 / 名 / フリガナ / フリガナセイ / フリガナメイ /
    ニックネーム / 性別 / 電話番号 / Email / 画像 / 指名（個人指名・男女指名）/ 個人指名料 /
    フリーメッセージ / 対応コース（チェックリスト）/ セット / 表示順序 / 給与情報
- **対応コース欄はコースマスタ（メニュー/コース名）から動的に引っ張る**設計にする（ハードコードしない）。
  コース紐づけは後で行うので、今はマスタのコース名から一覧生成だけ用意。
