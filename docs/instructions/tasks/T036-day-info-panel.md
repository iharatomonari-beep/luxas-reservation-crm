# タスクID：T036
# タスク名：当日情報パネル（左サイド）を3タブ化（予約情報/販売情報/返客情報）

## 安全レベル
CAUTION（予約台帳のUI追加）

## 目的
PMの台帳左サイド「当日情報一覧」に相当する3タブ（予約情報/販売情報/返客情報）パネルを整える。現状の「選択中の予約」パネルを拡張。

## 背景
catalog §1（当日情報一覧＝予約情報/販売情報/返客情報タブ・自店絞り込み・顧客検索）。LUXASは「選択中の予約」表示のみ。

## 対象ファイル / 変更してよいファイル
- `features/reservations/reservation-ledger.tsx`
- 必要なら `features/reservations/` に小コンポーネント

## 変更してはいけないファイル
- `features/customers/` `features/import-export/` `features/master-data/`
- `lib/supabase/` `supabase/` `.env*` `package.json` `tailwind.config.ts`、上記以外

## 実行内容
1. 左サイド（または台帳上部）にタブ: 予約情報 / 販売情報 / 返客情報。
2. 予約情報=当日の予約リスト（選択中ハイライト・クリックで詳細）。販売情報=当日の会計/物販（T022/T032があれば実値、無ければ枠）。返客情報=当日の返客/キャンセル（T026/T035連動）。
3. 顧客名のクイック検索（任意）。
4. 既存のドラッグ/カード操作を壊さない。レイアウトは行圧縮(T016)と両立。

## 検証方法
- `npm run lint` / `npm run build`／実機でユーザー確認。

## 完了条件
1. 3タブが切り替わり当日情報を表示。2. 既存を壊さない。3. lint/build OK。4. sessions追記・_index更新。5. 実機OKで完了。

## 停止条件
- レイアウト大改変が必要なら一旦タブ＋予約情報だけで止め報告。指定外ファイルはSTOP。lint/build失敗で停止。

## 完了後の報告内容
- 完了ID/追加タブと内容/lint・build/_index更新/実機確認依頼
