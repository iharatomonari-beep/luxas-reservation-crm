# タスクID：T050
# タスク名：返客一覧をPMの列に合わせる（種別/予約タグ/返客理由ステータス）

## 安全レベル
CAUTION（booking-returns.tsx の列調整＋返客区分の最小マスタ化）

## 目的
PMの返客一覧（返客情報）の列に合わせる。

## PMの仕様
- 検索: 日付範囲(from〜to) + 検索 + 検索条件追加
- 列: ID / 日付 / 開始時間 / 顧客名 / 種別 / 返客理由・キャンセル待ちステータス / コメント / 予約タグ

## 対象ファイル
- features/reservations/booking-returns.tsx
- 参照: types.ts（cancelType/cancelReason/bookingTagIds 等）

## 実行内容
- 列をPM順に: ID / 日付 / 開始時間 / 顧客名 / 種別 / 返客理由・ステータス / コメント / 予約タグ。
- 種別: cancelType（通常取消/無断キャンセル/返客 等）を表示。
- 返客理由・ステータス: cancelReason を表示（無ければ「-」）。返客専用区分マスタは将来。最小限、固定「キャンセル」をやめ実値表示に。
- 予約タグ: bookingTagIds から表示（無ければ「-」）。
- LUXAS独自の担当/コース列は残してよいが、PM列を優先配置。
- 上部に「検索条件追加」ボタン（最小UI）。

## 変更してはいけないファイル
- 上記以外。型の破壊変更はしない。

## 検証
- npm run lint / npm run build 成功。実機: 列がPM順、種別/理由/タグが実値（無ければ「-」）。

## 完了条件
1. 列PM準拠。2. 実値表示。3. lint/build OK。4. _index更新＋sessions追記。5. 実機OKで完了。
