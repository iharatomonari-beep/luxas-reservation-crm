# タスク名：予約受付のコース選択をPM風（カテゴリ色分けタブ＋コース一覧）に

## 安全レベル
CAUTION（予約作成フォームのコース選択UI変更。保存ロジックは不変）

## 目的
PMの予約入力はコースを「カテゴリ色分けタブ＋コース一覧（グリッド）」で選ぶ。LUXASは `<select>` ドロップダウン。これをPM風に変える。

## 対象ファイル
- features/reservations/reservation-ledger.tsx（台帳インライン作成フォーム）
- features/reservations/reservation-create-page.tsx（専用画面）
- 参照のみ: features/master-data/*（カテゴリ・サービス）

## 実行内容
- メニュー選択を、カテゴリ（services の category／カテゴリマスタ）でグループ化したタブ＋各タブ内のコース一覧ボタン（名前/料金/時間）に置き換える。選択中コースはハイライト。
- カテゴリの色は既存のluxasカラーを流用（新規lib追加なし）。
- 選択結果は従来どおり form.serviceMenuId に入れる（保存形式・自動終了時刻計算は不変）。
- 既存のオプション/割引/予約タグ/指名/連続などのUIはそのまま。

- 上記以外。型変更・依存追加はしない。
