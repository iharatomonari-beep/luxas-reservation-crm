# タスクID：T027
# タスク名：日次管理一式（出勤退勤/開店/レジ金点検/閉店/売上日報/月間目標/経費登録＋経費マスタ）

## 安全レベル
CAUTION（複数の新規画面・型追加。会計データ(T022)に一部依存）

## 目的
PMの日次管理相当の運用画面群を追加。会計(T022)が未了の集計はダミー/枠で進める。

## 背景
catalog §8,10。レジ金種テーブルは開店/点検/閉店で共通。売上日報は会計・予約・目標の集約。

## 対象ファイル / 変更してよいファイル
- 新規: `app/dashboard/daily/*`（attendance/open/register-check/close/report/expenses 等）
- 新規: `features/daily/*`（共通のレジ金種テーブル、各画面コンポーネント）
- 新規: 経費マスタ `features/master-data/cost-master.ts` + 画面
- `components/layout/sidebar.tsx`（導線）、月間目標は localStorage 新キー

## 変更してはいけないファイル
- `features/reservations/` の既存ロジック（会計はT022で実装済み前提で参照のみ）
- `features/customers/` `features/import-export/`
- `lib/supabase/` `supabase/` `.env*` `package.json` `tailwind.config.ts`、上記以外

## 実行内容（サブを順に・1つずつ）
1. 共通: レジ金種テーブル部品(10,000〜1円・本数/枚数→合計)。
2. 出勤・退勤: 当日シフト一覧＋打刻(出勤/退勤時刻を記録)。
3. 開店処理: レジ金入力→記録。レジ金点検: 金種入力→点検記録。閉店処理: 締め記録。
4. 売上日報: 本日の結果/来店数/予算達成度/明日の予算・予定（会計が無い項目は0/ダミー＋注記）。
5. 月間目標入力: 日付別 出勤人数/日次予算/総売上/コメント（T023と重複する場合は片方に寄せ報告）。
6. 経費マスタ(勘定科目/補助科目/表示順)＋経費登録(日付/科目/金額/摘要/何月分＋前月コピー)。
- 規模が大きいので、**サブ機能ごとに実装→lint/build→「実行中」→次サブ**。区切りでまとめて実機確認依頼。
