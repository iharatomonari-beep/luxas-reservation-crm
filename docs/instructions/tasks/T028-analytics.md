# タスク名：経営指標（集計レポート群）＋トップKPI＋物販販売

## 安全レベル
CAUTION（集計画面群。会計データ(T022)依存）

## 目的
PMの経営指標相当の集計を追加。会計(T022)の売上データを集計し、トップダッシュボードと各レポートに表示。

## 背景
catalog §E。売上・明細/スタッフ別/商品別/物販/時間帯別来店/日次集計/顧客数推移。LUXASトップは現状リンク集なのでKPI化。

## 対象ファイル / 変更してよいファイル
- `app/dashboard/page.tsx`（トップにKPIグラフ/集計）
- 新規: `app/dashboard/analytics/*` と `features/analytics/*`
- 新規: 物販販売 `features/reservations/` or `features/sales/`（物販明細）
- `components/layout/sidebar.tsx`（導線）

## 変更してはいけないファイル
- 既存の予約/顧客/マスタのロジック（参照のみ）
- `lib/supabase/` `supabase/` `.env*` `package.json` `tailwind.config.ts`、上記以外

## 実行内容（優先順・1つずつ）
1. トップKPI: 当月売上/前年比/達成度/来店(新規/リピート)。会計データから集計（無ければダミー＋注記）。
2. スタッフ別売上 / 商品別売上 / 時間帯別来店分析 / 顧客数推移 を順に。グラフは既存の描画手段（あるいは簡易テーブル）で可。
3. 物販販売明細（日付/顧客/物販/数量/単価/小計）。
- 会計が未実装の段階では各集計はダミー/0＋「会計実装後に実値」と注記。
