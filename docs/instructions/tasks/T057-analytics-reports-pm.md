# タスクID：T057
# タスク名：経営指標をPMの帳票群に拡充（＋CSV、グラフlibは追加しない）

## 安全レベル
CAUTION（経営指標画面の拡充。グラフlib追加はSTOP）

## 目的
PMの経営指標メニューに合わせる。

## PMの仕様（店舗）
- 売上・明細集計(ドリルダウン: 軸×指標) / スタッフ別売上 / 商品別売上 / 物販売上 / 時間帯別来店分析 / 帳票出力(各種リンク) / 日次集計(日別) / 日次集計(個人別) / クレジット会社別 / 顧客数推移。各 期間指定＋集計＋CSVダウンロード。

## 対象ファイル
- features/analytics/*（既存の analytics, analytics-reports を拡充）
- app/dashboard/analytics/*（必要に応じルート追加）
- components/layout/top-menu.tsx（経営指標グループに項目追加）

## 実行内容
- 既存の簡易集計を活かし、PMの軸（スタッフ別/商品別/物販/時間帯/日次/クレカ別/顧客数推移）を一通り用意。各に期間指定＋CSV出力（既存 serializeCsv 流用・lib追加なし）。
- 「帳票出力」はリンク集（店舗売上日報/実績明細/時間別来店数/個人実績/30日リピート/予約タグ集計/予約明細）として用意（中身は既存集計へ誘導 or 簡易表）。
- グラフは既存の簡易バー/表のまま。新規グラフlibは入れない（STOP）。

## 変更してはいけないファイル
- 会計・予約データの破壊変更。グラフlib等の依存追加（STOP）。

## 検証
- npm run lint / npm run build 成功。実機で各集計＋CSVダウンロードが動く＋メニュー導線。

## 完了条件
1. PMの主要帳票が揃う。2. CSV出力。3. lint/build OK。4. _index更新＋sessions追記。5. 実機OKで完了。

## 停止条件
- グラフlib/依存追加が要るなら STOP。会計依存が未整備なら該当帳票は0＋注記で進め報告。
