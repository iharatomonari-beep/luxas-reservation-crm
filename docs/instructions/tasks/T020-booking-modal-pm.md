- インターバル:ON
- 個別割引(% / 円) / 一括割引(% / 円)
- 連続
- 登録

## 実行内容（できる範囲を明示）
1. コース選択を**カテゴリ色分けタブ＋コース一覧ボタン**に変更（既存の `service.category` を使う。色は既存トーンで割当て）。選ぶと所要時間から終了時刻補完（既存ロジック）。
2. **こだわり**（希望なし/男性希望）トグルを追加（予約に保持。型に項目追加が要るなら最小で）。
3. **施術コメント**欄（既存メモを流用 or 追加）。
4. **個別割引/一括割引**（% と 円）入力欄を追加（保存はするが会計未実装なので表示・保持まで）。
5. **連続**トグル（枠だけでも可）。
6. **インターバル:ON**（既存があれば連動、無ければ表示のみ）。
7. **予約タグ・オプション商品**は、LUXASに対応マスタが無い。**勝手にマスタを作らず**、枠（プレースホルダ）に留めるか、必要なら「予約タグ/オプションのマスタ新設」を別タスク化する提案を報告して**停止**。
