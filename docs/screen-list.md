# 画面一覧

## 共通

### ログイン

Supabase Authでスタッフがログインする。未ログイン時は業務画面へアクセスできない。

### アプリシェル

左ナビまたは上部ナビで、予約台帳、時間グリッド、顧客、シフト、ブース、設定へ移動する。日付切り替えは予約系画面で共通化する。

## 予約

### 予約台帳

日別の予約を時刻順に表示する。顧客名、メニュー、担当スタッフ、ブース、ステータス、メモ有無を確認できる。

### スタッフ別時間グリッド

縦軸を時間、横軸をスタッフとして、予約、空き、シフト外を表示する。受付が当日の空き枠を素早く把握するための中心画面。

### 予約作成・編集

顧客、メニュー、担当スタッフ、ブース、日時、ステータス、メモを編集する。既存顧客検索から選択できる。

## 顧客

### 顧客検索

氏名、カナ、電話番号で検索する。予約作成中にも使えることを前提に、結果から顧客詳細または予約作成へ進める。

### 顧客詳細・カルテ

基本情報、注意事項、予約履歴、カルテメモを表示する。カルテ追記と顧客情報編集を行う。

## 管理

### スタッフ・シフト

スタッフ一覧と日別シフトを管理する。シフト外の予約は警告表示する。

### ブース管理

ブース名、説明、有効状態を管理する。予約と同じ時間帯でブースが重複する場合は警告する。

### メニュー管理

施術メニュー名、所要時間、価格、有効状態を管理する。v0.1では詳細なカテゴリ管理は行わない。

### 店舗設定

店舗名、営業時間、予約単位を設定する。single store前提のため店舗切り替えは作らない。

## 優先度

最優先は、ログイン、予約台帳、スタッフ別時間グリッド、予約作成・編集、顧客検索、顧客詳細・カルテ。シフト、ブース、メニュー、店舗設定はプロトタイプ運用に必要な最小機能に絞る。
