## 既知の問題・注意点
- **`useLocalCollection` には必ず安定参照（module定数の `initial*` か `EMPTY_*`）を渡す**。インライン `[]`/`{}` は無限ループ（Maximum update depth）を起こす。新規 collection を足すときは要注意。
- **dev サーバ稼働中に `npm run build` を実行しない**（共有 `.next` 破損で 500 / `Cannot find module './xxxx.js'`）。型確認は `npx tsc --noEmit`。
- 連続編集の途中状態で SWC（Next dev）が古いスナップショットのまま構文エラーをキャッシュすることがある。エラー行番号が現ファイルとズレていたら、`preview_stop → rm -rf .next → preview_start` でクリーン再起動。
- プレビュー（Claude管理）の `.click()` は React の onClick に届かない。`<a href>` の遷移は効くが、`<button>` のトグルは eval で検証不可。実機確認が必要。
- `.claude/settings.local.json`（個人設定）と `docs/reference/`（PM参考メモ・未追跡）は従来どおりコミット対象外。
